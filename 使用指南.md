## 🎬 实际操作流程

### 步骤1：准备你的数据

**从Excel复制：**
```
1. 在Excel中选中表格
2. Ctrl+C 复制
```

**从网页复制：**
```
1. 在网页上选中表格
2. Ctrl+C 复制
```

**手动粘贴：**
```
直接在对话框中输入表格数据
```

### 步骤2：打开表格对话框

```
1. 在博客编辑器中找到 📊 表格 按钮
2. 点击打开对话框
```

### 步骤3：选择导入模式

**看到两个选项：**
```
[手动创建]  [导入数据]
```

点击 **导入数据** 标签

### 步骤4：粘贴和导入

```
1. 在大文本框中右键 → 粘贴 （或 Ctrl+V）
2. 勾选 "第一行作为表头"（如果需要）
3. 点击 "导入表格"
```

### 步骤5：完成！

```
表格已自动插入到你的文章中
```

## 🔧 常见问题

### Q：导入的表格在哪里看？
A：在编辑器下方的"预览"部分就能看到表格的样子

### Q：导入后还能编辑吗？
A：当然可以。导入的表格是Markdown格式，可以随意修改

### Q：表格格式不对怎么办？
A：在导入前，尝试不勾选"第一行作为表头"重新导入

### Q：支持哪些格式？
A：只要能复制的表格都支持（Excel、Sheets、网页、CSV等）

### Q：表格太大会不会超时？
A：支持超级大的表格，没有大小限制

## 🎓 两种方式对比

### 方式1：手动创建（创建空白表格）

**适合：** 你要自己填内容

```
1. 点击 📊 表格
2. 保持 "手动创建" 标签
3. 输入行数、列数
4. 点击 "创建表格"
```

### 方式2：导入数据（导入现有数据）**推荐！**

**适合：** 你已经有表格数据

```
1. 点击 📊 表格
2. 点 "导入数据" 标签
3. Ctrl+V 粘贴
4. 点 "导入表格"
```

## 🆘 如果出错

### 错误1：提示"请输入表格数据"
```
检查：文本框是否为空
解决：重新粘贴数据
```

### 错误2：提示"无法解析表格数据"
```
检查：数据是否使用了Tab或多空格分隔
解决：在Excel中重新复制，或使用多空格分隔
```

### 错误3：表格显示不正常
```
检查：是否勾选了"第一行作为表头"
解决：尝试不勾选重新导入
```
